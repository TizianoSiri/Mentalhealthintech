# Mental Health in Tech - Data Analysis

## Introduction
In the tech industry, mental health is often overlooked despite its crucial impact on productivity and employee well-being. This project dives deep into survey data to uncover the workplace factors that impact mental health outcomes in the tech world.

## Project Features
- **Data Cleaning & Preparation**: Handling missing values, encoding categorical variables, and standardizing features.
- **Statistical Analysis**: Chi-square tests, logistic regression, and ordinal regression to identify key relationships.
- **Storytelling Insights**: Combining data analysis with a narrative to highlight human implications of mental health in tech.

## Repository Structure
- `data/`: Contains the survey dataset (`survey_data.csv`).
- `notebooks/`: Jupyter notebook with the full analysis (`analysis_notebook.ipynb`).
- `src/`: Python scripts for data processing and analysis.
- `docs/`: Full report and additional documentation.
- `LICENSE`: Licensing information.
